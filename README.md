# AI Agent Assistant

Multi-model AI Agent with tool calling, WebSocket streaming, and dark mode.

## Quick Start

```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```

Open frontend/index.html in browser.

## Features

- Multi-model switch (DeepSeek / MiMo / GPT-4o / Qwen / GLM-4)
- Tool calling (search / code / file / workflow)
- WebSocket real-time streaming
- Dark mode, responsive
- Plugin skill system
